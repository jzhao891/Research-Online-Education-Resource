XMLSpy XML Editor

Altova XMLSpy® 2017 is the industry's best-selling XML editor for modeling, editing, transforming, and debugging XML-related technologies. It offers the world’s leading graphical schema designer, a code generator, file converters, debuggers, profilers, full database integration, support for XSLT, XPath, XQuery, WSDL, SOAP, XBRL, JSON, and Office Open XML (OOXML) , plus Visual Studio and Eclipse integration, and more.

XML Editor

The XMLSpy XML editor abstracts away the complexity of working with XML technologies through its intuitive user interface and rich variety of XML editing views and options. Whether you prefer to develop in a text-based or graphical editing view, XMLSpy provides intelligent XML editing guidance as you type, and troubleshooting is fast and easy with the industry's most standards-conformant validation engine. More

XBRL Tools

XMLSpy provides powerful support for working with XBRL. The XMLSpy XBRL Validator can be used to validate both XBRL taxonomies and XBRL instance files to ensure accurate and compliant financial reporting. The graphical XBRL Taxonomy editor provides intelligent XBRL editing tools and an XBRL Taxonomy Wizard that walks you through extending standard XBRL taxonomies to meet specific requirements. More

XSL Tools

Editing and perfecting XSL stylesheets and XQuery code is finally easy. XSL and XQuery editors in XMLSpy provide intelligent entry helpers and info windows, and transforming your files is lightning fast. The XPath Analyzer makes it easy to generate or edit your XPath expressions, and powerful XSLT and XQuery debuggers and profilers take the guess work out of troubleshooting. More

Schema Tools

Support for complete schema development in XMLSpy includes a DTD editor, graphical XML Schema editor, and powerful schema generation and conversion wizards. XMLSpy will auto-generate a schema from an XML instance or a relational database structure, or you can model your own using the graphical design paradigm that supports drag-and-drop editing aided by intelligent entry helpers and validation.
